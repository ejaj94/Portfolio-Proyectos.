# Proyecto 10: Material de Branding "Com Cheiro de Amor"

Este proyecto contiene el diseño completo de los materiales gráficos complementarios para la marca "Com Cheiro de Amor®" (INPI Reg. 758901), enfocada en jabones botánicos, artesanales y veganos.

## Contenido del Proyecto

1. **Certificados de Participación y Mérito**: 
   - 5 modelos distintos (Premium Curvo, Botánico Ornato, Ondas de Aroma, Clásico Moldura Dourada y Minimalista).
   - Diseñados en HTML/CSS puros de alta definición, preparados para impresión en PDF A4 Horizontal.
   - Scripts de Node.js (`puppeteer`) para la generación automática de los PDFs en Inglés y Portugués.

2. **Tríptico Promocional y Hoja de Anotaciones**:
   - Diseño HTML/CSS de un panfleto tríptico a doble cara, destacando ingredientes naturales (Azeite, Óleo de Coco, Lavanda).
   - Hoja de anotaciones rústica a juego para los participantes del workshop.
   - Texturas botánicas y estética rústica (colores beige, terracota, blanco roto).

## Tecnologías Utilizadas

- **Frontend**: HTML5, Vanilla CSS3 (sin frameworks para máximo control de diseño e impresión)
- **Automatización**: Node.js, Puppeteer (para renderizado de PDF)
- **Tipografías**: Google Fonts (Cinzel, Playfair Display, Great Vibes, Montserrat)

## Instalación y Uso

1. Clonar el repositorio.
2. Navegar a la carpeta de `Certificados`.
3. Instalar las dependencias de Node:
   ```bash
   npm install
   ```
4. Generar los PDFs:
   ```bash
   node print_pdfs.js
   ```
