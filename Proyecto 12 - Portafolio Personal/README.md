- **Backend (Optional integration)**: Python, Flask, SQL (for dynamic APIs)

## Local Setup

### Running statically
Since the frontend is built with pure HTML/CSS/JS, you can simply open `index.html` in your browser.

### Running a local development server (Optional)
If you want to serve it locally using Python to prevent CORS issues with local assets:
```bash
# Optional: Create a virtual environment
python -m venv venv
source venv/Scripts/activate # Windows

# Install any requirements (if you plan to extend this into a Flask app)
pip install -r requirements.txt

# Serve statically via python
python -m http.server 8000
```
Then navigate to `http://localhost:8000`
