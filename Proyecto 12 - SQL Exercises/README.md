# Proyecto 12 - Ejercicios SQL

Este directorio contiene una colección de ejercicios prácticos de SQL, organizados por nivel de dificultad y tema, utilizando bases de datos de prueba (`sakila`, `world`) y bases de datos personalizadas (`encomiendas`).

## 📁 Archivos

*   `01_beginner_sakila.sql`: Ejercicios de nivel básico (SELECT, WHERE, ORDER BY, BETWEEN, LIKE) usando la base de datos `sakila`.
*   `02_intermediate_world_sakila.sql`: Ejercicios de nivel intermedio (INNER JOIN, multi-table JOIN, GROUP BY, HAVING, LIMIT) usando las bases de datos `world` y `sakila`.
*   `03_advanced_sakila.sql`: Ejercicios de nivel avanzado (Multi-table JOINs, GROUP BY + SUM/COUNT, Subqueries) usando la base de datos `sakila`.
*   `04_custom_encomiendas_db.sql`: Creación de una base de datos propia desde cero (`encomiendas`) con comandos DDL y DML (CREATE TABLE, INSERT INTO).
*   `05_world_db_languages.sql`: Ejercicios exploratorios usando la base de datos `world`, enfocados en la tabla `countrylanguage`.

## 🛠️ Tecnologías

*   MySQL / MariaDB
*   Bases de datos de prueba: Sakila, World
